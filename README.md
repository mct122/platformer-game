# Super Retro Platformer

このプロジェクトは、レトロな雰囲気のプラットフォーマーゲームです。
パソコンとスマートフォンの両方で遊ぶことができます。

## 遊び方

このゲームはブラウザで動作します。以下のURLにアクセスしてプレイしてください（ローカル開発環境の場合）。
`http://localhost:5173`

### 選択可能なキャラクター
- **Donko**: バランス型のキャラクター
- **Poon**: パワー型のキャラクター
- **Emanuel**: スピード型のキャラクター

### 操作方法

#### PC（キーボード）
- **移動**: 矢印キー (`←` / `→`) または `A` / `D` キー
- **ジャンプ**: 上矢印キー (`↑`) または `Space` キー

#### スマートフォン（タッチ操作）
- **移動**: 左下の `←` `→` ボタン
- **ジャンプ**: 右下の `Jump` ボタン
- **音声**: 右上のスピーカーアイコンでON/OFF切り替え

## 開発環境のセットアップ

Node.jsがインストールされている必要があります。

1. **依存関係のインストール**
   ```bash
   npm install
   ```

2. **開発サーバーの起動**
   ```bash
   npm run dev
   ```

## プロジェクト構造

主なソースコードは `src` ディレクトリにあります。

- **`src/engine/`**: ゲームのコアエンジン
  - `Game.js`: ゲームループ、状態管理、入力処理を担当するメインクラス
- **`src/entities/`**: ゲーム内のオブジェクト（エンティティ）
  - `Player.js`: プレイヤーキャラクターの挙動
  - `Koopa.js`: 敵キャラクター（カメ）の挙動
  - `Item.js`: アイテム（キノコなど）の挙動
- **`public/`**: 静的アセット（画像、音声）
- **`index.html`**: エントリーポイント
